## Valid paths
Given a set of paths and a source and a destination in a bidirectional graphs, find whether the source can reach the destination.

## Find all paths
Given a DAWG and a source and a destination, find all paths from the source to the destination.

## Connecting cities with minimum cost
Given a list of cities and a list of connections between cities, find the minimum cost to connect all cities.

## Find longest cycle
Given a graph, find the longest cycle in the graph.
